# GlassQuick Brand Guidelines

This document defines the visual identity of GlassQuick. All new features, components, and pages MUST follow these guidelines to maintain a premium and consistent user experience.

## Color Palette

### Core Brand Colors
| Color | Hex | Role | Variable |
| :--- | :--- | :--- | :--- |
| *Primary (Brand Deep Blue)* | #1A4F7A | Brand identity, logos, key accents | --primary |
| *Secondary (Brand Cyan)* | #0992C2 | Supporting brand elements, secondary accents | --secondary |
| *CTA (Glass Blue)* | #1E88E5 | Buttons, links, active menu, icons (Old Primary) | --cta |
| *Accent (Premium Teal)* | #00A8A8 | Offers, highlights, hover effects | --accent |
| *Accent Hover* | #1565C0 | Button hover effect (Dark Blue) | --accent-hov |

### Functional Colors
| Color | Hex | Role | Variable |
| :--- | :--- | :--- | :--- |
| *Success* | #2E7D32 | Success messages | --green |
| *Error* | #D32F2F | Error messages | --error |
| *Border* | #E0E0E0 | Input boxes, cards | --border |
| *Surface* | #FFFFFF | Cards, modals, button text | --surface |

### Typography Colors
| Color | Hex | Role | Variable |
| :--- | :--- | :--- | :--- |
| *Primary Text (Charcoal)* | #1A1A1A | Headings and important text | --text |
| *Secondary Text (Dark Grey)*| #555555 | Descriptions and normal text | --text-2 |
| *Light Text (Light Grey)* | #888888 | Helper text, placeholders | --text-3 |

## Typography


### 1. Poppins
- *Use Case*: Headings, navigation, buttons.
- *Weights*: 400, 500, 600, 700.
- *CSS Variable*: --font-sans

### 2. Inter
- *Use Case*: Body text, product descriptions.
- *Weights*: 400, 500.
- *CSS Variable*: --font-body

---

## ✨ Design Elements

### Shadows
- *Shadow MD*: 0 4px 20px rgba(0,0,0,.10) — Used for product cards and floating menus.

### Border Radius
- *Standard*: 8px or 12px for a softer, premium feel.
